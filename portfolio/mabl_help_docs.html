Mabl help docs

Support & help documentation website design

Overview

Background

The mabl help site is a main source of documentation for the application. The site has articles for users learning build tests in mabl, viewing the lastest release notes, and guides to troubleshoot issues. User's on this site can search for documentation by category or article name. In addition, they can submit a ticket to support to request help.

Problem

The mabl help site was in need of a redesign of homepage, categories, and article page. The old site was poorly outdated. It showed no resemblance to the company site's market branding colors or typography.

Wireframing

I led the visual redesign and development of the help documentation site. During my process, I collaborated with the marketing team to ensure the homepage and internal page designs met branding standards.
